# Job Posting: Full-Stack Engineer (TypeScript/Python)

## About PulseAI IDE Code Editor
PulseAI IDE is an AI-native software engineering platform built on VS Code OSS. Our mission is to become the most capable AI development environment by enabling developers to build, test, debug, and deploy software through intelligent AI agents. We aim to automate the entire software development lifecycle while keeping developers in full control.

## Position Overview
We are seeking a talented Full-Stack Engineer with expertise in TypeScript/JavaScript and Python to join our founding team. You will be responsible for building and maintaining the core IDE platform, integrating AI capabilities into VS Code OSS, and developing backend services for model management and agent orchestration.

## Key Responsibilities
- Develop core AI chat and agent interfaces within the IDE using TypeScript/JavaScript and VS Code extension APIs
- Implement file system integration, terminal, and debugging features
- Build backend services for model management and agent orchestration using Python (FastAPI)
- Collaborate with AI Engineer to integrate language models and agent frameworks
- Ensure performance, stability, and quality of the IDE
- Participate in code reviews, sprint planning, and daily standups

## Required Qualifications
- Strong proficiency in TypeScript/JavaScript and VS Code extension development
- Proficiency in Python and backend development (FastAPI preferred)
- Experience with Language Server Protocol (LSP), debug adapters, or similar technologies
- Familiarity with AI model integration (APIs or local models)
- Experience with UI/UX implementation (React/Tailwind is a plus)
- Strong problem-solving skills and ability to work independently and collaboratively
- Excellent communication skills and a proactive attitude

## Preferred Qualifications
- Experience with agent frameworks (LangChain, LlamaIndex, etc.)
- Knowledge of MCP (Model Context Protocol)
- Background in developer tools or IDEs
- Familiarity with DevOps practices and CI/CD pipelines
